**Abstract**: Un poliedro puede ser pensado como un mont&oacute;n de pol&iacute;gonos que se pegan unos con otros a trav&eacute;s de sus aristas. Esta noci&oacute;n es puramente combinatoria y se extiende de manera natural a dimensiones superiores de manera recursiva, es decir, podemos pensar a los pol&iacute;gonos como objetos 2-dimensionales y para n > 3, hablar de objetos n-dimensionales construidos a partir de objetos (n - 1)- dimensionales. Este problema se vuelve retador cuando ciertas condiciones de simetr&iacute;a son impuestas. En la pl&aacute;tica exploraremos esta situaci&oacute;n y hablaremos brevemente sobre una construcci&oacute;n de este estilo para ciertas teselaciones del toro.

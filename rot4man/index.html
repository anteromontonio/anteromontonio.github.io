Census of rotary 4-maniplexes

This is a companion website for the census of rotary 4-maniplexes (Cunningham, Mochán, Montero, Potočnik, Toledo, 2025). The census is available in the following Zenodo repository.

Cumulative growth of rotary 4-maniplexes

Cumulative counts of rank-4 maniplexes vs. n, the number of flags. Hover a chart for exact values at a given n; use "table" to see the underlying rows.

P — Polytopal F — Faithful U — Unfaithful O — Orientably regular N — Non-orientable (regular) C — Chiral
x-axis (n)
y-axis (cumulative count)

By maniplex class — P / F / U breakdown

For each of the three classes of maniplexes (Orientably regular, Non-orientable, Chiral), the cumulative counts of Polytopal / Faithful / Unfaithful maniplexes, plus the class total.

By P / F / U — O / N / C breakdown

For each of Polytopal / Faithful / Unfaithful, the cumulative counts split by maniplex class (O / N / C), plus the total. R4M-P/F/U have cumulative formulas computed only up to n = 5000 in the source workbook (vs. n = 10000 for R4M-O/N/C) — their curves stop earlier for that reason, not because the underlying census does.
